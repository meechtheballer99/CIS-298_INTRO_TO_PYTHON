NOTE ON STRINGS: [ : : ] == [:], first value is start, next is stop, last is step count and direction, where direction defines relative beginning;
	+ direction means start is relative to beginning and moving through to the end,
		 and - direction means start is relative to the end and goes through to the beginning.
